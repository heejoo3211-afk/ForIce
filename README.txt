파일명은 전부 영어이며 디스코드에 보이는 명령어만 한국어입니다.

최상위 명령어 12개:
가방
구매
메뉴판
mc구매
mc적립
mc지갑
포인트
메뉴
콜팅
매니저
설정
패널

추가된 기능:
- /설정 충전메시지채널
- /패널 전송
- /패널 새로고침
- 포인트 충전/제거 메시지를 큰 글씨로 공개 출력
- 충전 메시지는 설정한 전용 채널로 출력
- 패널에는 포인트 수령 버튼 없음
- 기존 JSON 데이터는 삭제하지 않음

설치:
1. 기존 data 폴더를 유지하려면 users.json, settings.json, menu.json을 새 폴더 data 안에 복사
2. .env 작성
3. npm install
4. npm run deploy
5. npm start

성공 출력:
명령어 등록 완료: 12개
명령어 로드 완료: 12개
